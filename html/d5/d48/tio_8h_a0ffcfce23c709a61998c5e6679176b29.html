unsigned char tioGetDefC ( const char *  name,
const unsigned char  Default 
)

Функция получения параметра в виде одного символа

Функция возвращает параметр символ переданный после параметра с указанным именем. Если в качестве параметра передан не один символ а строка возникает ошибка. Как и при любой другой ошибке функция в этом случае возвращает значение default.

Аргументы:
name имя параметра
Default значение возвращаемое при ошибке
Возвращает:
Символ в случае успеха, default при ошибке.

Перекрестные ссылки tio_param_rec::has_key, tio_param_rec::name, TEINCTYPE, TENOPAR, TENOTSET, TESUC, Tio_params, Tio_params_sz, tioGetDefC() и tio_param_rec::val.

Используется в tioGetDefC().
