tioDebug
tioDebugF
int tioDebug ( const char *  message  ) 

Функция вывода отладочной информации

Данная функция исполняется только если приложению через командную строку передан параметр --tio-debug. Если такой параметр не передан приложению то программа тутже возвращает управление вывзвающему приложению не производя никаких действий с переданным сообщением.

Аргументы:
message Текст сообщения.
Возвращает:
0 при успешном завершении и код ошибки в противном случае.

Перекрестные ссылки TEFAIL и tio_self_state.
