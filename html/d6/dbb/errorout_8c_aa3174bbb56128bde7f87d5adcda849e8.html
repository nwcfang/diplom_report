int tioError ( const char *  content  ) 

Передает в поток ошибок текст message.

Аргументы:
content Строка предназнаенная для вывода
Возвращает:
0 при успешном завершении и код ошибки в противном случае

Перекрестные ссылки TEFAIL.

Используется в tioDie().
