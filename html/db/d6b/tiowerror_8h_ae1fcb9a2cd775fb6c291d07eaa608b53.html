int tioSetErrorNum ( int  num  ) 

Установка кода ошибки для последующего вызов tioGetError()

Аргументы:
num Код ошибки
См. также:
tioGetError(), tioerror.h
Возвращает:
0 при успешном завершении TEINTMC при попытке использовать неинициализированную очередь ошибок (вызов tioSetErrorNum до tioErrorInit) и TEEPICFAIL в случае ошибок по памяти.
