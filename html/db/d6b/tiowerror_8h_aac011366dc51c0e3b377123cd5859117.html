int tioErrorInit ( void   ) 

Функция инициализации работы с ошибками

Данная функция инициализирует внутренние структуры для многопоточной обработки ошибок. При этом каждому из потоков будут возвращены ошибки возникше именно в нем.

Возвращает:
Функция возвращает 0 при успешном выполнении TEINTMC при попытки повторной инициализации и TEEPICFAILL в случае возникновения ошибок в памяти
