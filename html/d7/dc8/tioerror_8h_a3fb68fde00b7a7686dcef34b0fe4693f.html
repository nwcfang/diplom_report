int tioGetError ( void   ) 

Получение кода ошибки для последней вызванной из потока функции библиотеки.

Возвращает:
Код ошибки или в случае если используется ненинициализированая очередь ошибок TEEPICFAIL.

Перекрестные ссылки cmppthread_t() и tioGetError().

Используется в tioGetError().
